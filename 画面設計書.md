## 補足情報

### 共通コンポーネント
- **ヘッダーコンポーネント**: `result-header`クラスで共通化されており、SHIPロゴ、タイトル、件数表示、アクションボタンで構成
- **フィルターヘッダー**: `filter-header`クラスで実装され、各画面で絞り込み条件を指定
- **ナビゲーションメニュー**: `nav-menu`クラスでドロップダウンメニューを実装

### 使用ライブラリ
- **Choices.js**: セレクトボックスの拡張機能（検索、複数選択など）
- **html5-qrcode**: QRコード読み取り機能
- **ExcelJS**: Excel出力機能

### 画面遷移フロー
```
ログイン画面
  ↓
メイン画面（資産検索）
  ├→ 資産検索結果画面
  │   ├→ 資産詳細画面
  │   ├→ 申請一覧画面
  │   │   └→ 見積依頼一覧画面
  │   └→ 見積書管理画面
  ├→ QRコード管理画面
  │   ├→ QR詳細画面
  │   ├→ QR発行画面
  │   └→ QR印刷画面
  └→ マスタ管理
      ├→ SHIP施設マスタ画面
      └→ SHIP資産マスタ画面
```

## Excel転記時の注意事項

1. **表のコピー方法**: 各表はMarkdown形式なので、Excel貼り付け前にテキストエディタでパイプ（|）区切りをタブ区切りに変換すると便利です
2. **スクリーンショット**: 各画面のスクリーンショットは別途撮影し、「画面基本情報」の下に挿入してください
3. **項目の追加**: 実装が進んだ場合は、対応する画面の「UI要素一覧」に行を追加してください
4. **フォーマット**: Excelでは「画面基本情報」と「UI要素一覧」を別シートにするか、同一シート内で区切って管理することを推奨します
